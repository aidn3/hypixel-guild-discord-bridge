# Security Policy

## Supported Versions

This repository is [Rolling Release](https://wikipedia.org/wiki/Rolling_release).
That means only the latest commit on `master` branch is supported.

## Reporting a Vulnerability

Either contact me via the Discord server in [README.md](./README.md) or via e-mail at security (at) aidn5.com.

Please explain how the vulnerability works and provide a demo like a video evidence to showcase the vulnerability.
After confirming the vulnerability, I will:

- patch the vulnerability in the source code
- roll out the hot fix to all publishers (like Docker images, git release tags, etc), so people can receive the update as soon as possible
- contact all users/hosters I know of to notify them to update as soon as possible
- create an announcement urging everyone to update while disclosing the vulnerability
